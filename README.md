# Premaluu Devtools

## How do I install these formulae?

`brew install premaluu/devtools/<formula>`

Or `brew tap premaluu/devtools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "premaluu/devtools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
